as the file exists. Creating the shim file IS the cutover.

Ground rules (apply to every task)
----------------------------------

    If a needed signature cannot be inferred from repo call sites, STOP and

    (restir_di_*.hlsli, restir_gi_*.hlsli); shims go at the exact
    Rtxdi/DI/... or Rtxdi/GI/... path. Keep the RTXDI_* public names used
    by call sites so no call-site edits are needed.

    neo/shaders/CMakeLists.txt (mirror how the restir_pt_* files are
    listed), and remove the corresponding ${RTXDI_RUNTIME_DIR} entries
    from the DI list (~lines 108-117) / GI list (~lines 128-135) once the
    in-repo shim exists.

R5. Tasks tagged MATH: do not attempt the derivation. Create the file
    skeleton if instructed, then stop and report which task is blocked.

R6. After each completed task, append a one-line entry to the Progress
