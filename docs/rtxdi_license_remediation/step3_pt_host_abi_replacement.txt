Notes
-----

The packed PT reservoir remains an opaque C++ stride placeholder. Shader-side
reservoir packing, unpacking, address calculation, resampling math, path-tracer
context, and random-context logic remain separate replacement modules.

The host reservoir pitch helper intentionally mirrors the current shader-side
tiled addressing formula used by the remaining PT shaders:

    blockCountX = ceil(width / 16)
    blockCountY = ceil(height / 16)
    reservoirBlockRowPitch = blockCountX * 16 * 16
    reservoirArrayPitch = reservoirBlockRowPitch * blockCountY

This is compatibility only. Replacing the shader reservoir/addressing module is
still required before the PT path is license-clean.
