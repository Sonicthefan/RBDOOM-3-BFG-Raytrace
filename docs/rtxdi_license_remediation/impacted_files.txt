Impacted File Inventory
=======================

Every file that directly consumes proprietary NVIDIA RTXDI headers
(SPDX LicenseRef-NvidiaProprietary, NVIDIA RTX SDKs License), plus the
build files that wire the include path and the artifacts that contain
compiled NVIDIA code. Audit date 2026-06-11.

"Action" key:
    REWRITE  rebuild against rbdoom-owned replacement headers
    DELETE?  legacy lane; prefer deletion over rewrite (plan Step 3)
    EDIT     remove/replace the dependency wiring
    REBUILD  binary artifact; regenerate after source remediation

Legacy PT / smoke lanes  -  neo/shaders/builtin/pathtracing/
------------------------------------------------------------

These lanes consume Rtxdi/PT/* (ReSTIR PT) headers from the upstream
RTXDI sample - a basic non-shipping example. The project's mandate was to
conform to the RTX Remix shipping design; using the PT sample files
directly likely degraded performance and quality on top of the license
issue. This strengthens the case for DELETE? over REWRITE.
