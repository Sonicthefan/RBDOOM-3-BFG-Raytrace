    E:\prog\references\RTXDI-main\Samples\FullSample\Source\RenderPasses\LightingPasses\LightingPasses.cpp

Reference math and function shape:

    RTXDI_SampleLightsForSurface:

        1. computes RTXDI_InitialSamplingMisData from
           RTXDI_DIInitialSamplingParameters

        2. samples local lights through RTXDI_SampleLocalLights

        3. RTXDI_SampleLocalLightsInternal selects lights through
           RTXDI_SelectNextLocalLight

        4. UniformSampling.hlsli selects a local index from
           RTXDI_LightBufferRegion and calls RAB_LoadLightInfo(lightIndex,false)

        5. RTXDI_StreamLocalLightAtUVIntoReservoir calls
           RAB_SamplePolymorphicLight, computes blended source PDF through
           RTXDI_LightBrdfMisWeight, computes target PDF through
           RAB_GetLightSampleTargetPdfForSurface, and streams the sample into a
           DI reservoir

        6. local sampling finalizes with:

               RTXDI_FinalizeResampling(state, 1.0, misData.numMisSamples)
               state.M = 1

        7. RTXDI_SampleLightsForSurface combines local/infinite/environment/BRDF
           reservoirs with RTXDI_CombineDIReservoirs, then finalizes with:

               RTXDI_FinalizeResampling(state, 1.0, 1.0)
               state.M = 1

        8. if enableInitialVisibility is set and the selected DI reservoir is
           valid, it calls RAB_GetConservativeVisibility and stores visibility
           in the DI reservoir

    FullSample path tracer NEE shape:

        SampleDirectLightsForIndirectSurface calls RTXDI_SampleLightsForSurface.
        CalculateNEE then uses the selected DI reservoir and selected
        RAB_LightSample to compute:

            radianceFromLights =
                RAB_GetReflectedBsdfRadianceForSurface(...) *
                visibility *
                RTXDI_GetDIReservoirInvPdf(diReservoir) /
                lightSample.solidAnglePdf

            sampledLightData.lightData = diReservoir.lightData
            sampledLightData.uvData = diReservoir.uvData
            neePdf = 1.0 / diReservoir.weightSum
            scatterPdf = RAB_SurfaceEvaluateBrdfPdf(...)

    LU-02A must not reimplement or modify that math. It should only call the
    RTXDI helper and display the resulting reservoir/sample diagnostics.
