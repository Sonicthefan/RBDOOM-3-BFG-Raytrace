ReSTIR PT External RAB Input Inventory
Date: 2026-05-17

Purpose
-------
List every input that enters the local RAB bridge from outside the RAB callback
files, grouped by current condition.

    docs\ReSTIR\restir_pt_rab_io_supply_map.txt

Definition
----------
"External to RAB" means the value is supplied by rbdoom shader resources,
rbdoom CPU builders, RTXDI PT pass parameters, RTXDI PT reservoir state, or
the NVIDIA FullSample application bridge. If the current rbdoom RAB file says
its supplier is another RAB file, this inventory names the ultimate external
supplier behind that RAB-to-RAB edge.

"RTXDI supplier" means the reference-side file or sample bridge that shows the
expected input shape. Many inputs are not supplied by the RTXDI library itself:
the library calls RAB callbacks and expects the application bridge to provide
engine-specific G-buffer, material, geometry, light, environment, and PSR data.
